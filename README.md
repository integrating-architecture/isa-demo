###ISA Apps for the Enterprise
are a new and innovative technology, that provides **platform independent modularity** and **communication** for new and existing enterprise systems without additional costs. 

eApps enable the evolutionarily build of distributed enterprise applications from lightweight and versioned service modules, that are located in a central repository and provided on demand at runtime as modular application systems, central business rules and universal services hosted by a distributed Service Broker.
